Creating Program Tokens

  1. First click[left] your program name.

  2. Go to My Tokens tab

  3. Types of token fields located in right sidebar.

    • Text Field Example: Token for Name, Company, Location and Day.
    • Date Field Example: Calendar Date.
    • Image Field Example: Image Url.
    • Number Field Example: Final Count.
    • Rich Text Field Example: A WYSIWYG Text Editor.
    • Calendar Field Example: Calendar Invitation.

  4. To Create Tokens just DRAG and Drop the selected Field in Local Token container.

  5. The First Column is for Token Name and the Second Column is for the Token Content.

  6. We have Default Token Names should follow:

    • Should start in 'What this Token For?' and Always Start with Capitalized Letter followed by dash (-) and what should the token property.
      • EMAIL example: {{ Email - From Name}} {{ Email - Header}}
      • LANDING PAGE example: {{ Lp - Header}}
      • EVENTS example: {{ Event - Title}}

    • Event - Title

      Event - Time Start

      Event - Time End

      Event - Time Zone

      Event - Date

      Event - Day

      Event - Location

    • Email - From Name

      Email - From Title HEADLINES.(*uppercase).... <_span_style="_font-family:_Arial,_Helvetica,_sans-serif;_color:#636169;_"_> FORUM

      Email - Sender Address marketingops@spglobal.com

      Email - StrapLines

    • Ical - Body (Calendar Hyperlink for Body)

      Ical - CTA (Calendar Hyperlink for Call to Action Button)

  7. In Creating Calendar Token Follow This steps..

    • DRAG AND DROP THE Calendar Token Field to Local Tokens Container.
    • Use proper name for Calendar Token Field we have default token name for Calendar Tokens.

      Calendar Token for Body Default = Ical - Body

      Calendar Token for Call To Action (CTA) Default = Ical - Cta

    • For this example we will create Ical - cta.
    • After you put the Token name ('Ical - Cta')Click to Edit and Edit Calendar File will show.
    • And now go to yourTask Docx/ Program Docx File and go to Calendar Appointment Copysection.
    • Reflect the Calendar Appointment Copy Content to your Calendar Token.

      NOTEToken is not applicable for Subject, Location,HyperLink text, Starts and Ends input fields.

      NOTEUsed Existing token for Description field.

      In Hyperlink text you should wrap the Content to span with styles see the example: (Wrap in Span is just for cta only not in body calendar) for the color of text is #ffffff[white] text-decoration is none<_span style="color:#ffffff; text-decoration: none;">ADD TO CALENDAR<_/span>

      For time zone Always Check & Convert the time zone example: IST to SGT.

    • then Click Save.
